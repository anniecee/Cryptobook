## Getting started:
- Set up client:
```
cd client
npm i -g yarn
npm i
yarn start
```

- Set up server:
```
cd server
npm i
npm run dev
```

## Running:
- Run client:
```
yarn start
```
- Run server:
```
npm run dev
```
